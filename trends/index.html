    __init__(self, keywords, timeframe='all', geo='', cat=0, gproperty='') special ¶

    Create the Google Trends client with basic parameters.

    Parameters:

    Name Type Description Default
    keywords List[str]

    A list of keywords or one keyword.

    		 required
    timeframe str

    Time range of the query. Allowed patterns are 'all', '%Y-%m-%d %Y-%m-%d', '%Y-%m-%dT%H %Y-%m-%dT%H', '5y', '1m', '3m', '12m', '1d', '7d', '1h', '4h'. Defaults to 'all'.

    		 'all'
    geo str

    Two letter country abbreviation like 'US' or more specific like 'US-CA'. Defaults to '' meaning worldwide.

    		 ''
    cat int

    Category id of the query. Use GoogleTrends.categories() to get all category names and ids. Defaults to 0 meaning all categories.

    		 0
    gproperty str

    Property section of the query. Can be 'images', 'news', 'youtube', 'froogle'. Defaults to '' meaning Google Search.

    		 ''
